FEATURED PROJECTS
Anime Store Hotspot & LISA

Analyzed spatial patterns of anime stores in Tokyo using LISA for population clustering, Near Table proximity analysis, and Getis-Ord Gi* hotspot detection. Automated workflows with ArcPy and integrated Google Places API data revealed that stores cluster around high-density areas and cultural hubs like Akihabara, Ikebukuro, and Shibuya.

Elderly Access to Services

Analyzed geographic access to long-term care facilities for Tokyo’s elderly population using ward-level density, GIS overlay, and ArcPy automation. Choropleth maps, facility location overlays, and accessibility analysis revealed well-served central wards and gaps in peripheral, low-density regions, providing insights for municipal planning and improving equitable access to services.

NC Working Age Lisa Analysis in R

Analyzed geographic access to long-term care facilities for Tokyo’s elderly population using ward-level density, GIS overlay, and ArcPy automation. Choropleth maps, facility location overlays, and accessibility analysis revealed well-served central wards, while some high-density areas like Toshima lacked facilities despite proper infrastructure, providing insights for equitable LTC planning.
